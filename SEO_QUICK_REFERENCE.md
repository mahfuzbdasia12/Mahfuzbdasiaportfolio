## On-Page SEO Checklist (Per Page)

### Before Publishing

**Meta Tags**:
- [ ] Title: 50-60 characters, keyword in first 3 words
- [ ] Description: 150-160 characters, keyword naturally included
- [ ] H1: Single, keyword-rich, descriptive
- [ ] Keywords meta: Added (if applicable)

**Content Quality**:
- [ ] 1,500+ words for blog posts
- [ ] Unique, original content (not copied)
- [ ] Natural keyword distribution (1-2%)
- [ ] Proper heading hierarchy (H1→H2→H3)
- [ ] Read time disclosed
- [ ] External sources cited

**Images & Media**:
- [ ] All images have descriptive alt text
- [ ] Images compressed (optimized file size)
- [ ] Image titles added
- [ ] At least 1 featured image (1200x630px)
- [ ] Images include keywords in alt text

**Internal Linking**:
- [ ] 3-5 internal links per page
- [ ] Links use keyword-rich anchor text
- [ ] Links point to relevant pages
- [ ] No broken links
- [ ] Links open in same tab (not new window)

**Technical**:
- [ ] Canonical URL set
- [ ] Robots meta tag: index, follow
- [ ] Page loads in <3 seconds
- [ ] Mobile responsive
- [ ] Schema markup added (where applicable)

## Core Web Vitals Optimization

### Monitoring Dashboard

**Target Scores**:
- LCP (Largest Contentful Paint): < 2.5s
- FID (First Input Delay): < 100ms
- CLS (Cumulative Layout Shift): < 0.1

**Current Status** ✅:
- LCP: ~1.4s (Good)
- INP: ~50ms (Good)
- CLS: Near-zero (Good)

### Quick Fixes for Common Issues

**Slow LCP**:
1. Preload critical fonts
2. Optimize hero image (size, format)
3. Defer non-critical CSS
4. Use CDN for assets

**High INP**:
1. Reduce JavaScript execution time
2. Break up long tasks
3. Use web workers for heavy computations
4. Throttle event handlers

**High CLS**:
1. Add explicit width/height to images
2. Avoid inserting content above existing content
3. Use CSS transform animations
4. Reserve space for ads/embeds

## Troubleshooting Guide

### Issue: No Organic Traffic

**Possible Causes**:
- [ ] Site not indexed (check GSC)
- [ ] Noindex tag accidentally added
- [ ] Robots.txt blocking indexation
- [ ] Too new (requires time)
- [ ] Keyword competition too high

**Solutions**:
- Request indexing in GSC
- Check meta robots tag
- Verify robots.txt
- Target lower-competition keywords
- Build more backlinks

### Issue: Traffic Drop

**Possible Causes**:
- [ ] Algorithm update
- [ ] Manual action applied
- [ ] Technical issues (site speed, indexation)
- [ ] Competitor outranking
- [ ] Seasonal variation

**Solutions**:
- Check Google Search Console alerts
- Review GSC Coverage report
- Run technical audit
- Analyze ranking changes
- Update content quality

### Issue: Poor Core Web Vitals

**Possible Causes**:
- [ ] Large unoptimized images
- [ ] Heavy JavaScript
- [ ] Render-blocking resources
- [ ] No caching
- [ ] External scripts slow

**Solutions**:
- Optimize images (WebP, compression)
- Code splitting and lazy loading
- Async/defer JavaScript
- Enable caching headers
- Use Content Delivery Network (CDN)
