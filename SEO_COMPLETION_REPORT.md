**LocalBusiness Schema** (Local SEO targeting):
```json
{

  "url": "https://mahfuzbdasia.com",
  "email": "mahfuzbdasia@gmail.com",

}
```

**ProfessionalService Schema** (Service offering markup):
```json
{

  }

```
